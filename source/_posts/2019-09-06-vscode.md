项目通常会用`@/`来做为别名映射路径 `src/`, 通过添加 `jsconfig.json` 文件，来唤起路径提示。

最后更新时间 2019-09-06

<!-- more -->

在项目根目录下，添加一个 jsconfig.json 文件

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    },
    "target": "ES6",
    "module": "commonjs",
    "allowSyntheticDefaultImports": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}
```
